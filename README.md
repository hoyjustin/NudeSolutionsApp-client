# Nude Interview Client-App

This project is the client side submission for the Nude Solutions developer assignment.
A table displays customers' high-value items for contents limit insurance.

** This was bootstrapped using [Create React App](https://github.com/facebook/create-react-app).


## Available Scripts

In the project directory, you can run:

### `npm start`

Runs the app in the development mode.\
Open [http://localhost:3000](http://localhost:3000) to view it in the browser.

### `npm run build`

Builds the app for production to the `build` folder.\
It correctly bundles React in production mode and optimizes the build for the best performance.
